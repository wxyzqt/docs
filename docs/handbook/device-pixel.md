# 设备像素

## 物理像素（Physical Pixels）

设备屏幕上实际存在的物理像素点数量。

例如：一台 1080x1920 分辨率的手机，其屏幕上有 1080 列和 1920 行的物理像素点。

## 逻辑像素（CSS 像素 / Device Independent Pixels, DIP）

逻辑像素是开发者在 CSS 或前端代码中使用的像素单位。

逻辑像素与物理像素的比例由设备像素比（DPR, Device Pixel Ratio）决定。

例如：在一台 DPR 为 2 的设备上，1 个逻辑像素等于 2x2 个物理像素。

## 设备像素比（DPR, Device Pixel Ratio）

表示物理像素与逻辑像素的比例。

常见的 DPR 值：

1：标准分辨率设备（如老式显示器）。

2：高分辨率设备（如 Retina 屏幕）。

3：超高分辨率设备（如部分 4K 手机屏幕）。

公式：DPR = 物理像素 / 逻辑像素

## 屏幕分辨率

表示屏幕的物理像素宽度和高度。

常见分辨率：

720p：1280x720（HD）

1080p：1920x1080（Full HD）

1440p：2560x1440（2K）

2160p：3840x2160（4K）

## PPI（Pixels Per Inch）

每英寸的像素数量，表示屏幕的像素密度。

PPI 越高，屏幕显示越清晰。

常见 PPI：

低 PPI：100-150（普通显示器）。

中等 PPI：200-300（普通手机屏幕）。

高 PPI：300+（Retina 屏幕或 4K 显示器）。

## 常见设备像素分类

标准分辨率设备：DPR = 1，例如老式显示器。

高清设备：DPR = 2，例如 iPhone 6、7、8。

超高清设备：DPR = 3，例如 iPhone 11 Pro、部分 Android 旗舰机。
